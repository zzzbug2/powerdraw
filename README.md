# Powerdraw

A small shell script to view various system info on x86 systems.

## Dependencies
```
lm-sensors
acpi
```
## Output
```
User
Date
Uptime

Tasks
Threads
RAM in use

Battery Charge
Battery Watts drawn
Battery Voltage

Fan speed

CPU Temp
GPU Temp

```

## Usage

```
chmod +x powerdraw
./powerdraw
```
