# Powerdraw

A small shell script to view various system info on x86 systems.

## Dependencies
```
lm-sensors
acpi
```
## Output
```
Date
Uptime

Total Tasks/Threads (including kthr)

Who

RAM in use
RAM Free

SWAP in use
SWAP Free

Battery Charge
Battery Watts drawn
Battery Voltage

Fan speed

CPU Temp
GPU Temp

```

## Usage

```
chmod +x powerdraw
./powerdraw
```
