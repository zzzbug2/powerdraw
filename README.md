# Powerdraw

A small shell script to view various system info on x86 systems.

![Picture of powerdraw running](https://1rc.ca/images/powerdraw.png)

## Dependencies
```
lm-sensors
acpi
```
## Output
```
Date
Uptime

Who

Top 5 output from top

Total Tasks/Threads (including kthr)

Total CPU Load

RAM in use
RAM Free

SWAP in use
SWAP Free

Battery Charge
Battery Watts drawn
Battery Voltage

Fan speed

CPU Temp
GPU Temp

CPU Frequencies

```

## Usage

```
chmod +x powerdraw
./powerdraw

```
